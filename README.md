## Getting Started 🚀

### ⚙️ Local Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Inspectra-Cyber-Team/Inspectra-UI.git
   ```
2. Install Dependencies
   ```sh
   npm i or npm install
   ```
3. Run the project
   ```sh
   npm run dev
   ```
